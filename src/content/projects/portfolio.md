## Le projet en quelques mots

Mon portfolio pour présenter mon travail de développeur web. Inclut un "mode éco" pour démontrer une approche alternative applicable aux sites d'information et de service.

## Particularités du projet

- **Transitions persistantes** : Les éléments communs ne se rechargent pas entre les pages
- **Web sans cookies** : Aucun bandeau de consentement en restant dans la légalité
- **Double mode** : Version standard et version éco accessibles via URLs distinctes
- **Redirection intelligente** : Détection automatique de la langue du visiteur

Conçu pour être exemplaire en termes d'éco-conception et de performance, chaque choix technique vise l'efficacité.

## Project in a few words

My portfolio, built to showcase my work as a web developer. It includes an "eco mode" to demonstrate an alternative approach, applicable to informational and service-based sites.

## Project highlights

- **Persistent transitions** : Shared elements don't reload between pages
- **Cookie-free** : No consent banner, while staying fully compliant
- **Dual mode** : Standard and eco versions, accessible via separate URLs
- **Smart redirection** : Automatic detection of the visitor's language

Designed to set an example in eco-design and performance, every technical choice aims for efficiency.

## Projekt in wenigen Worten

Mein Portfolio, das meine Arbeit als Webentwickler zeigt. Es enthält einen "Öko-Modus", der einen alternativen Ansatz demonstriert, anwendbar auf Informations- und Service-Websites.

## Projektbesonderheiten

- **Persistente Übergänge** : Gemeinsame Elemente werden zwischen den Seiten nicht neu geladen
- **Cookie-frei** : Kein Einwilligungsbanner, und dennoch rechtskonform
- **Doppelmodus** : Standard- und Öko-Version über getrennte URLs erreichbar
- **Intelligente Weiterleitung** : Automatische Erkennung der Besuchersprache

Entwickelt, um in Sachen Öko-Design und Leistung beispielhaft zu sein – jede technische Entscheidung zielt auf Effizienz.
